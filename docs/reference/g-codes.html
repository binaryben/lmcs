2  G-Code cheatsheet

Knowing G-code commands will help you use the MDI to control the movement and operation of the CNC spindle. Understanding what each command does can also be helpful when debugging .nc files that have been created by your postprocessors.

  • Machinist Guides: Complete G-Code List

2.1 Common operations

Here are some examples of using the most common g-codes to achieve simple operations:

G01 X100 Y100 Z20 F5000

2.2 G-Codes

Code Result

G02

 CW arc move

G03

 CCW arc move

2.5.1 G-Codes

Full list of supported G-codes can be found here on the Masso website.

Code Result

G00

 Do not use (Rapid movement)

G04

 Dwell (pause)
G17 to G19 Plane selection (G17 is the default, do not change)

G20

 Do NOT put the machine in inches!

G21

 Coordinates in metric
G40 to G43 Tool compensation commands

G53

 Cancel work offsets
G54 to G59 Work offsets

G90

 Absolute positioning

G91

 Relative positioning (Do NOT use)
